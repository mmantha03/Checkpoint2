# Wilderlight Forager

## Concept

Wilderlight Forager is a 3D wilderness survival-collection game. The player explores a forest, gathers supplies, lights a campfire, and escapes through the exit gate before time or health runs out.

## Objective

Collect six supplies, return to the campfire, light it, then follow the green exit marker and press E at the gate to win.

## Controls

- WASD: Move
- Mouse: Move camera
- Space: Jump
- Left Shift: Sprint
- E: Collect / interact
- R: Reset camera or restart after ending
- Escape: Release mouse

## Gameplay Systems

- Collectible wood, food, and crystal resources
- Inventory/resource tracking
- Health and countdown timer
- Campfire interaction
- Locked exit gate and win condition
- Healing shrine
- Hazard zones
- Patrolling enemy watchers
- Win/lose screen with restart flow
- Objective, inventory, health, and timer UI
- Ambience, footsteps, pickup sounds, damage sounds, and campfire audio

## Known Issues

- Visuals use a simple low-poly style.
- Enemy behavior is basic patrol/chase movement.
- The project still uses Unity's classic Input Manager warning.
